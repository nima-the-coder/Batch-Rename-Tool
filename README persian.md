# Rename machine
ابزار تغییر نام دسته‌ای فایل‌ها
یک ابزار پایتونی برای تغییر نام خودکار و گروهی فایل‌ها با رابط کاربری گرافیکی.

     ویژگی‌ها
-رابط کاربری گرافیکی (GUI) با Tkinter

- انتخاب آسان پوشه با دیالوگ استاندارد

- پشتیبانی از پیشوند دلخواه کاربر

- شماره‌گذاری خودکار (001، 002، 003...)

- پیش‌نمایش تغییرات قبل از اجرا

- گزارش کامل از عملیات انجام شده

- مدیریت خطا و ادامه عملیات در صورت خطا


    فناوری‌های استفاده شده
Python 3.8+

Tkinter - رابط گرافیکی

OS Module - مدیریت فایل و پوشه

datetime - تولید گزارش با تاریخ
